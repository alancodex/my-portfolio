Html

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Css

CSS (Cascading Style Sheets) é uma linguagem usada para definir a aparência e o layout de páginas web. Ela permite que você separe o conteúdo (HTML) do design visual, controlando aspectos como cores, fontes, espaçamento, tamanhos e posicionamento de elementos.



Nível: Intermediario

⇡
JavaScript

JavaScript é uma linguagem de programação que permite criar interatividade e dinamismo em páginas web. Enquanto o HTML define a estrutura e o CSS define a aparência, o JavaScript controla o comportamento dos elementos na página.



Nível: Intermediario

⇡
Bootstrap

Bootstrap é um framework front-end que facilita o desenvolvimento de websites e aplicações web responsivas e de design moderno. Ele combina HTML, CSS e JavaScript, oferecendo um conjunto de componentes prontos e estilos pré-definidos que ajudam a acelerar o processo de criação de interfaces visuais.



Nível: Intermediario

⇡
Node.JS

Node.js é um ambiente de execução JavaScript que permite executar código JavaScript no servidor, fora do navegador. Ele é baseado no motor V8 do Google Chrome e é amplamente utilizado para criar aplicações web escaláveis e de alto desempenho.



Nível: Intermediario

⇡
Python

Python é uma linguagem de programação de alto nível, conhecida por sua simplicidade, legibilidade e versatilidade. Ela é amplamente utilizada em diversos domínios, como desenvolvimento web, ciência de dados, automação, inteligência artificial, entre outros.
